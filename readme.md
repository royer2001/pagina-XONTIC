# Xontic

Página web corporativa de Xontic, una empresa de desarrollo de software a medida.

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Estructura

- `index.html`: Página principal.
- `style.css`: Estilos, diseño responsivo y modo oscuro.
- `men.js`: Funcionalidad del formulario y cambio de tema.

## Características

- Diseño responsivo.
- Modo oscuro (Dark Mode).
- Formulario de contacto funcional (simulado).

## Instalación

Simplemente abre `index.html` en tu navegador web.
